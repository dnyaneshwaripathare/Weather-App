# Weather-App
# 🌤️ Weather App

A responsive Weather App built using **HTML**, **CSS**, and **JavaScript** that provides real-time weather information using the **WeatherAPI**.

## 📌 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition
- 💧 Displays humidity
- 🌬️ Shows wind speed
- 📱 Responsive and user-friendly interface
- ⚡ Real-time weather data using WeatherAPI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- WeatherAPI
